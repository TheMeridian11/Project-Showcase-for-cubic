# Project Showcase for Cubic

This repo is a small showcase of projects I have worked on across robotics, C++, Java, and machine learning. Each one is a bit different, but they all show something I have been learning or experimenting with.

## Cargo Robotics

An Arduino-based cargo robot project. The robot is designed to pick up cargo at one point, drive toward another point, avoid obstacles with an ultrasonic sensor, and release the cargo using a servo.

**Main ideas:**

* Arduino motor control
* Ultrasonic obstacle detection
* Servo cargo release
* Simple state-based robot logic

## DSA

A C++ data structures and algorithms project focused on weighted directed graphs. It includes graph handling, negative cycle checking, and all-pairs shortest path algorithms like Johnson’s algorithm and Floyd-Warshall.

**Main ideas:**

* C++ templates
* Graph algorithms
* Shortest paths
* Test cases using small and random graphs

## Java_shop-main

A JavaFX shop / warehouse style app. It has users, suppliers, products, carts, orders, controllers, views, and model classes. I used this project to practise building a more structured Java app with separate parts for the UI and the actual logic.

**Main ideas:**

* JavaFX
* FXML views
* Controllers and models
* Basic app structure

## VectorParticleSystem

A C++ visual physics project using raylib. It shows particles moving with forces like gravity, wind, and friction. The visual trails and force arrows make the vector math easier to understand instead of just looking at formulas.

**Main ideas:**

* C++ graphics
* Vector math
* Particle movement
* Gravity, wind, and friction simulation

## ML PNT Project

A Jupyter notebook about targeted adversarial attacks on an MNIST classifier. It uses FGSM and PGD to slightly change digit images and test how the model reacts. It also includes Gaussian smoothing as a simple defence idea.

**Main ideas:**

* Python / PyTorch
* MNIST classification
* Targeted FGSM and PGD attacks
* Basic adversarial ML defence testing

## Notes

This repo is not meant to be one big finished app. It is more like a collection of work that shows different areas I have explored, from hardware and algorithms to GUI apps and machine learning.
